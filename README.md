# Media Finder

A web app for finding what to watch next written with Rust, React, and TypeScript.

![Media Finder](./media-finder.png)

## Building

### Front End

```bash
npm run build --prefix ./client
```

### Back End

```bash
cargo build
```

## Running

### Front End

```bash
npm run dev
```

### Back End

```bash
cargo run
```
